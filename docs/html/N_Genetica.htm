PopulationTProgram, TOutput
Represents a simple implementation of IPopulationTProgram composed of ITreeProgramTOutput. The algorithm follows a traditional evolutionary algorithm to step one generation:
  1. performs selection to get a pool of n parents for crossover, where n is the size of the population; performs crossover from parent pool to get some offspring (given percentage of the population); performs mutation from parent pool to get some mutated programs (given percentage of the population); performs elite selection (keeps a given percentage of the population corresponding to the best programs); creates some random programs (given percentage of the population).

Genetica.Operators.Mutation Namespace

 
Classes
  ClassDescription
FitnessSimplifyMutation
Represents a IMutationOperatorTProgram for MathProgram. This mutation operator tries to simplify (shorten the expression of) a given program by removing descendant programs that do not affect its fitness by some degree.
HoistMutationTProgram, TOutput
Represents a IMutationOperatorTProgram for ITreeProgramTOutput. This operator selects a random sub-program of a given program.
PointMutationTProgram, TOutput
Represents a IMutationOperatorTProgram for ITreeProgramTOutput. Allows the mutation of programs by randomly replacing each sub-program by one program with the same arity taken from some PrimitiveSetTProgram.
ShrinkMutationTProgram, TOutput
Represents a IMutationOperatorTProgram for ITreeProgramTOutput. This mutation operator removes a random descendant node of a given program and replaces it with a random terminal program from a given PrimitiveSetTProgram.
SimplifyMutation
Represents a IMutationOperatorTProgram for MathProgram. This mutation operator simplifies (tries to shorten the expression of) a random descendant program of a given program.
StochasticMutationTProgram
Represents a generic IMutationOperatorTProgram that selects from a list of mutation operators at random.
SubtreeMutationTProgram, TOutput
Represents a IMutationOperatorTProgram for ITreeProgramTOutput. Represents a mutation operator that replaces one sub-program of a given program by a new random program generated using some IProgramGeneratorTProgram, TOutput.
SwapMutationTProgram, TOutput
Represents a IMutationOperatorTProgram for ITreeProgramTOutput. This operator mutates a given program by swapping (reversing the order of) the children of a randomly-selected function sub-program.

IMutationOperatorTProgram
An interface for mutation operators for IProgramTInput, TOutput, i.e., operators that take one program and create a new one by changing some sub-program in a certain way.
